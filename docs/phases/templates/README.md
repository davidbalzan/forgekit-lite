# Planning Templates

Templates used by `/plan-phase` to generate task documents.

## Available Templates

| Template                  | Purpose                                |
| ------------------------- | -------------------------------------- |
| `TASK_TEMPLATE.md`        | Output format for phase task documents |
| `task_template_prompt.md` | AI prompt for guided task planning     |

## Usage

These templates are used automatically by the `/plan-phase` skill. You can also copy them manually to plan tasks.
