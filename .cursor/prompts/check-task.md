# Check Task - Progress Tracker

Mark tasks as complete in phase task files and update progress.

## Instructions

1. Find the task file: `docs/phases/phaseN/PHASEN_TASKS.md`
2. Locate the task by number or description
3. Update checkbox: `- [ ]` to `- [x]`
4. Update progress stats if present
5. Report what was marked complete and remaining tasks
